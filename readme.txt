一、相关技术
1、lombok---用于pojo注解get与set的包，除了maven加依赖后，
   还需要在idea安装插件
2、AdminLTe 后端框架
3、springboot dubbo 测试只需要在测试类上
	@RunWith(SpringRunner.class)
	@SpringBootTest(classes = XbinStoreServiceAdminApplication.class)
	类不是新建，而是注入的方式
4、mybatis 生成代码
5、druid数据库连接池
   密码没有加密，设置config.decrypt=false
6、Swagger2构建强大的restful api文档
7、页面引擎采用 Beetl 
8、网页采用freemarker生成静态化页面
9、SpringBoot项目在IntelliJ IDEA中实现热部署
10、SpringBoot 自定义Starter
